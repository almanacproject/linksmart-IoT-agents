Detailed Description

Provides the application program interface (API) necessary to control the Physical World Adaptation Layer (PWAL) drivers and expose underlying variables, services and events through it.

The PWAL API is composed of:

  • An annotations package (see with the provided types of PWALAnnotations A driver package (see with several interfaces needed to interact with the PWALDriver An exceptions package (see with supported types of PWALExceptions/code>s A model package (see with the different PWALObjects used by the PWAL.
