1. INSTALL plugin RAP

Use either market place or this following link for latest stable builds

eclipse Software site: http://download.eclipse.org/rt/rap/tools/2.3 

2. INSTALL the RAP TARGET

http://eclipse.org/rap/developers-guide/devguide.php?topic=target-install.html&version=2.2
Using the RAP Tools to install the latest target platform

On the Eclipse welcome screen (Help > Welcome), go to Overview > Remote Application Platform. 
Choose Install Target Platform on the following page. 
In the upcoming dialog confirm the default values, which starts the target installation process.

3. Import the PWAL-GUI from git

Run the file pwal-gui.launch -> run as -> pwal-gui
